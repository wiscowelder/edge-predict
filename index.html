⚙️ About & Model Info

How it works: brain.py runs on GitHub's servers via GitHub Actions. Training happens once (30–90 min). Daily updates take a few minutes. This page reads data.json and shows the results — no calculations happen here.

API keys are stored as GitHub Secrets — never in any file.

To train: GitHub repo → Actions → "Train Foundation" → Run workflow
To update predictions: GitHub repo → Actions → "Daily Update" → Run workflow
Auto-updates: Predictions refresh daily at 6 AM UTC automatically.
🔑 CBS & March Madness Button Setup

The "Generate Picks" and "Generate Brackets" buttons trigger GitHub directly from this page. To enable them, add a github_config block to your data.json:

"github_config": { "owner": "YOUR_GITHUB_USERNAME", "repo": "YOUR_REPO_NAME", "token": "ghp_YOUR_PERSONAL_ACCESS_TOKEN" }
To get a token: GitHub → Settings → Developer Settings → Personal Access Tokens → Fine-grained tokens.
Permissions needed: Actions: Read & Write on your repo only.

⚠️ Keep this file private (your repo should be private, which it is if you're self-hosting).
