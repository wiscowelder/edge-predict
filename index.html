Overview

EDGE Predict is a machine-learning sports prediction system running on GitHub Actions — no server costs. Predictions update daily at 6 AM CT automatically.

The model trains on historical data from 1999–present using XGBoost with 30+ signals per sport. Primary goal: win CBS NFL Confidence Pool and March Madness brackets.

Setup: GitHub Config

The "Generate" buttons need your GitHub token to trigger workflows. Create a file called config.json in the root of your repo (NOT data.json — that file is too large to edit in GitHub's web editor):

{ "github_config": { "owner": "YOUR-GITHUB-USERNAME", "repo": "edge-predict", "token": "github_pat_YOUR_TOKEN_HERE" } }

⚠️ Never share your token in chat or commit it publicly. Keep the repo private.

API Keys (GitHub Secrets)

Add these in your repo → Settings → Secrets → Actions:

BALLDONTLIE_KEY — Required. Free at balldontlie.io ODDS_API_KEY — Recommended. Free at the-odds-api.com OPENWEATHER_KEY — Optional. Free at openweathermap.org

Workflows

Train Foundation — Run manually after updating brain.py. Takes 60–90 min. Required after first setup and after major model changes.

Daily Update — Runs automatically at 6 AM CT. Also triggerable via the Predictions tab button above.

CBS Picks — Generate before Thursday Night Football kickoff each week. Re-generate any time before a game starts — locked games are preserved.

March Madness — Available after Selection Sunday. Generate once, then re-generate after each round completes.
